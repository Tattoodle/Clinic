# Rathmines Doctors Clinic

Website for Rathmines Doctors Clinic (Dr George Joyce), a GP practice at
104 Lower Rathmines Road, Dublin 6.

## Structure

- **`clinic-app/`** — the current site. Vite + React, react-router, framer-motion.
  Multi-page (Home, Services, Meet the Doctor, Contact), soft-tone medical design,
  intro loader + page transitions, SEO/structured data.
- **`website/`** — earlier static HTML/CSS version, kept for reference.

## Run

```bash
cd clinic-app
npm install
npm run dev
```

Dev server runs on http://localhost:5184 (or pass `-- --port <n>`).
